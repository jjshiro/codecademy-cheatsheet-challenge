How to add a form to your webpage

1. Create your form using the <form> tag.
2. Choose your User input:

Input

Tags & Attributes

Example

<labelfor="x">
Password Text Field <input type="password" name="x" id="x">
<label for="x">
Number <input type="number" name="x" id="x" step="1">
<label for="x">
Slider <input type="range" name="x" id="x" min="0" max="0" step="1">
<label for="x">
<span>

3. Add optional attributes for form validation:

Attribute

Description
required Makes an input field compulsory - the form cannot be submitted without it.
min & max Sets a minimum and/or maximum limit on numerical input.
minlength & maxlength Sets a minimum and/or maximum limit on text length.
pattern Restricts the number and/or kinds of characters that will be accepted using a regex.
